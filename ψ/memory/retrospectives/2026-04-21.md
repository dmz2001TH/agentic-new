## เรียนรู้ (Learnings)
- **WSL-Windows Bridge**: การคุยข้ามฝั่ง OS ผ่านพอร์ต Network มักติด Firewall แต่การรันสคริปต์ผ่านตัวรันฝั่ง Windows (`node.exe`) คือทางลัดที่ทรงพลังที่สุด
- **Resilient Web Scanning**: การรอหน้าเว็บโหลดต้องไม่รอจนอินเทอร์เน็ตนิ่ง (networkidle) แต่ต้องรอแค่ "เป้าหมายปรากฏ" (selector-based) เพื่อความเร็วและลด Timeout
- **Memory Consistency**: การบันทึกความผิดพลาด (Mistake Memory) ช่วยให้เอเจนท์ฉลาดขึ้นอย่างรวดเร็วในการแก้ปัญหาเดิม
