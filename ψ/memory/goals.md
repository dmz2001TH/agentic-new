# เป้าหมาย (Goals)

## วิธีใช้
ทุก agent ต้องติดตามเป้าหมายในไฟล์นี้ สร้าง goal ใหม่เมื่อมี task ใหญ่
อัพเดท progress ระหว่างทำงาน ปิด goal เมื่อเสร็จ

## สถานะ Goal
- `[ ]` = ยังไม่เริ่ม
- `[~]` = กำลังทำ
- `[x]` = เสร็จ
- `[!]` = ติดปัญหา (ต้องการความช่วยเหลือ)
- `[-]` = ยกเลิก
